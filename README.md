# A Delta Robot Simulator in Python

## To run as a dash app:
### Using poetry, run:
    - poetry install
    - poetry shell
    - python app/app.py
### Using pip, run:
    - pip install -r requirements.txt
    - python app/app.py

***
### To modify the robot structure
set the params accordingly and press `reset`
